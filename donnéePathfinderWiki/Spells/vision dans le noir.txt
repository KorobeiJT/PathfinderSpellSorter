{s:Desambi|Vision dans le noir est également le nom d'une vision dans le noir|capacité spéciale.}
[École] Transmutation
[Niveau] alchimiste 2, antipaladin 2, ensorceleur/magicien 2, rôdeur|Rôd 3
[Temps d'incantation] 1 action simple
[Composantes] V, G, M (une pincée de carotte séchée ou une agate)
[Portée] contact
[Cible] créature touchée
[Durée] 1 heure/niveau
[Jet de sauvegarde] Volonté, annule (inoffensif)
[Résistance à la magie] oui (inoffensif)

La cible du sort acquiert la faculté de voir jusqu'à dix-huit mètres (12 ) de distance dans le noir le plus complet. Il voit les formes normalement, mais en noir et blanc.

Il est possible d'user de ''permanence'' sur un sort de ''vision dans le noir''.
